# Master - YT

Overview:-

This project mimics the core features of YouTube which includes:-

1. Video Streaming
2. Video Playback
3. Home Feed
4. Video Suggestion and related Videos
5. Search Results and Search Feed
6. Likes and Views count

Deployed Link:- 
https://master-yt.netlify.app/
